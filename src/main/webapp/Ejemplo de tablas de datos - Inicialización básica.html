Inicializacion basica

Editor es una extensión Crear, Leer, Actualizar y Eliminar (CRUD) para las Tablas de datos que brinda la capacidad de agregar, editar y eliminar filas fácilmente en una base de datos que se muestra en una Tabla de datos. Editor proporciona una interfaz limpia y sensible para la manipulación de datos por parte del usuario final, una API expresiva para un control completo y un protocolo de comunicaciones del servidor bien definido para el envío de datos.

Este sencillo ejemplo muestra una tabla con siete campos, cada uno de los cuales puede editarse como texto sin formato. En otros ejemplos, exploraremos cómo agregar selectores de fecha, selectelementos y otros controles para que la entrada de formularios sea intuitiva para el usuario del sistema, entre muchos otros aspectos de la API del Editor.

/*
 * Example PHP implementation used for the index.html example
 */
 
// DataTables PHP library
include( "../../php/DataTables.php" );
 
// Alias Editor classes so they are easy to use
use
    DataTables\Editor,
    DataTables\Editor\Field,
    DataTables\Editor\Format,
    DataTables\Editor\Mjoin,
    DataTables\Editor\Options,
    DataTables\Editor\Upload,
    DataTables\Editor\Validate,
    DataTables\Editor\ValidateOptions;
 
// Build our Editor instance and process the data coming from _POST
Editor::inst( $db, 'datatables_demo' )
    ->fields(
        Field::inst( 'first_name' )
            ->validator( Validate::notEmpty( ValidateOptions::inst()
                ->message( 'A first name is required' ) 
            ) ),
        Field::inst( 'last_name' )
            ->validator( Validate::notEmpty( ValidateOptions::inst()
                ->message( 'A last name is required' )  
            ) ),
        Field::inst( 'position' ),
        Field::inst( 'email' )
            ->validator( Validate::email( ValidateOptions::inst()
                ->message( 'Please enter an e-mail address' )   
            ) ),
        Field::inst( 'office' ),
        Field::inst( 'extn' ),
        Field::inst( 'age' )
            ->validator( Validate::numeric() )
            ->setFormatter( Format::ifEmpty(null) ),
        Field::inst( 'salary' )
            ->validator( Validate::numeric() )
            ->setFormatter( Format::ifEmpty(null) ),
        Field::inst( 'start_date' )
            ->validator( Validate::dateFormat( 'Y-m-d' ) )
            ->getFormatter( Format::dateSqlToFormat( 'Y-m-d' ) )
            ->setFormatter( Format::dateFormatToSql('Y-m-d' ) )
    )
    ->process( $_POST )
    ->json();
